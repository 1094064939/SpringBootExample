# SpringBootExample

##SpringBoot项目工程

本项目会持续跟新

### - 目录列表

①quickstart项目
- 使用devtools进行热部署
- 通过Webjars引用静态资源
- 添加Thymeleaf依赖

②spring-data-jpa

- 里面有两个例子，一为book展示界面，另一个为student类，为JPA的起步练习
- 使用JPA描述对象－关系表的映射关系，并将运行期的实体对象持久化到数据库中


③spring-data jpa进阶

- 通过解析方法名创建查询进行操作
- 仿"极简图床"，对图片进行分类，并放到相对应的"相册"里，点击相册，跳转页面，展示对应的图片

④timing

- 仿微信步数排行榜，实现登录进入步数排行榜界面，现实所有成员的运动数据，并在每天某个时间定时刷新步数
